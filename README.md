# 📱 Flutter Swippable Card Animations  

## 🚀 Features  
- 🔄 **Swipe left/right animations**  
- 🎯 **Customizable card designs**  
- 🛠️ **Adjustable animation speed and direction**  
- 📏 **Stacked deck effect**  
- 🔥 **Gesture detection for swipe actions**
